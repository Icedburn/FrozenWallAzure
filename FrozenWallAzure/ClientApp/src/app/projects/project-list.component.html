My Projects =]

In this page you will find a list of my projects. Well... Actually this page itself is the first project listed, so it is a good example on how it works. On every line you will find a GitHub link for the project and a link to see how it works. In each project page you will find a small explanation of the project, like the paragraph below and a way to see the project working.

Although it's a simple page, I decided to build it on Angular to study/play with it. Also it 100% hosted on AWS. Yes, a completely overkill, but hey! You can find a nice "Show Last Update" button to show or hide some data of the table. Also a filter that you can filter the projects by their name or language (or both). So, if you want to see only Go language project, just filter by "Go" :)
